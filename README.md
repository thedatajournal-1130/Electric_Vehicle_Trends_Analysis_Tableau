# 🚗Electric_Vehicle_Trends_Analysis_Tableau
# 📌 Project Overview
This project analyzes electric vehicle (EV) adoption trends to uncover insights related to growth patterns, regional distribution, and market dynamics.
Using Tableau, the dataset was visualized into an interactive dashboard to identify key trends in EV adoption and support data-driven insights into the evolving automotive landscape.

# 🎯 Objectives
● Analyze growth trends in electric vehicle adoption

● Identify regional distribution of EV usage

● Understand category-wise performance (BEV vs PHEV)

● Track changes in adoption over time

● Provide insights into market expansion opportunities

# 🧰 Tools & Skills Used
● Tableau

● Data Visualization

● Dashboard Design

● Data Analysis

● Trend Analysis

● Storytelling with Data

# 📂 Dataset Description

The dataset includes information related to electric vehicles, such as:

Vehicle Type (BEV / PHEV)

Registration Year

State / Region

Make & Model

Electric Range

CAFV Eligibility

# 📊 Key Analysis Performed

► Year-wise EV adoption trends

► Regional distribution of EVs

► Comparison between BEV and PHEV vehicles

► Manufacturer and model-level analysis

► Range-based vehicle performance

# 🔍 Key Insights

◆ EV adoption shows a strong upward trend over recent years

◆ Battery Electric Vehicles (BEVs) dominate over Plug-in Hybrid Vehicles (PHEVs)

◆ Recent years show accelerated adoption 

◆ A few manufacturers account for a significant share of EV registrations
